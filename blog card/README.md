## My process

### Built with

* Semantic HTML5 markup
* CSS Custom Properties (Variables)
* CSS Flexbox (for centering layouts and internal component alignment)
* Mobile-first workflow

### What I learned

During this project, I deepened my understanding of CSS layout axes and debugging spacing inconsistencies.

1. **Flexbox Coordinate Alignment:** I mastered how `justify-content` controls the main X-axis alignment, while `align-items` acts as the vertical control for the cross-axis (Y-axis), allowing me to perfectly center components both on the page and relative to each other.
